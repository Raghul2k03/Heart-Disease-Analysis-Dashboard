# Heart-Disease-Analysis-Dashboard
 ❤️ Heart Disease Analysis Dashboard | Power BI Project

1. Project Introduction
The Heart Disease Analysis Dashboard is a Power BI project designed to analyze patient health records and identify important factors affecting heart disease survival rates.  

This dashboard converts raw healthcare data into interactive visual insights that help in understanding patient conditions, survival trends, and risk patterns across different age groups.

---

 2. Project Objective
The main goals of this project are:

. To analyze heart disease patient data efficiently  
. To monitor survival and death statistics  
. To identify high-risk health conditions  
. To compare survival rates across age groups  
. To create an interactive healthcare analytics dashboard  

---

4. Dashboard KPIs
The dashboard includes the following key performance indicators:

- Survival Rate
- Average Age of Survival
- Total Survival Count
- Total Death Count

---

 5. Dashboard Visualizations

5.1 Survival Analysis by Age Group
Displays the total survival count for different age categories.

5.2 Serum Creatinine Analysis
Shows the average serum creatinine level across age groups.

5.3 Ejection Fraction Analysis
Analyzes the average ejection fraction values by age category.

5.4 Survival Rate Trend
Visual representation of survival percentage trends among patients.

5.5 Health Risk Factor Analysis
Shows the impact of:
- Smoking
- Diabetes
- Anaemia
- High Blood Pressure

on patient survival.

---

8. Key Insights

1. Middle-aged patients show higher survival rates.  
2. Patients above 71 years have lower survival percentages.  
3. Smoking and high blood pressure significantly affect patient outcomes.  
4. Serum creatinine levels vary notably across age groups.  
5. Survival patterns can be better understood using interactive visual analytics.
